===========================
How to use 1-my_list.py
===========================

This module defines a class My_List that inherists form the class list.

Usage
-----

Returns an ascending sorted list.

First import ``My_List`` from the ``1-my_list.py`` file:

    >>> MyList = __import__('1-my_list').MyList

Tests:

::
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

::
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(-4)
    >>> my_list.append(2)
    >>> my_list.append(-3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, -4, 2, -3, 5]
    >>> my_list.print_sorted()
    [-4, -3, 1, 2, 5]

::
 
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []
